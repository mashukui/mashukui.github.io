如何选择和使用

汇总首页负责产品导航，详细功能和使用限制以各产品页与当前版本说明为准。

01

按平台选择

先确定目标平台和数据类型，再进入对应产品页查看支持的采集模块。

02

查看产品说明

产品页提供界面截图、使用流程、授权价格、常见问题和合规说明。

03

下载当前版本

客户端统一通过各项目 GitHub Releases 发布，Windows 与 macOS 支持以发布说明为准。

合规使用说明

这些工具仅用于合法合规的公开数据整理、学习研究和自有业务场景。使用者应遵守目标平台服务协议、隐私政策和所在地法律法规，不得用于高频破坏性请求、个人敏感信息交易或其他侵犯平台及用户合法权益的行为。

从目标平台开始选择

进入产品页查看功能边界、软件界面、当前价格和版本下载。
